Floko

Bring cool new features and a clean experience to your Android device

Documentation GitHub

Latest Android

Floko is released at least once a month. Updates include latest security patch, and new features. Now Floko "p9.0" is based on Android 9.0 "Pie".

Clean and Fast

Floko is based on AOSP, the Android Open Source Project. There is no bloatware, spyware, or annoying notifications from OEMs.

Customizable

Floko has many customization features such as SmartBar, Fling, Pulse, Network activity indicator, OmniSwitch, 4x4/5x5/6x6 Lock pattern...
