Latest Android

FlokoROM is released at least once a month. Updates include latest security patch, and new features. Now FlokoROM v3 is based on Android 10.

Clean and Fast

FlokoROM is based on AOSP, the Android Open Source Project. There is no bloatware, spyware, or annoying notifications from OEMs. Taste clean Android Experience.

Customizable

FlokoROM v3 has many customization features. such as Network activity indicator, OmniSwitch Recents, Transparency and so on.
