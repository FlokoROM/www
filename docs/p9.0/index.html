History

In 2017, we started project called "AICP-mordiford". It's a fork of AICP (Android Ice Cold Project), and provide its UNOFFICIAL build.
We worked on n7.1(nougat) and o8.1(oreo) release. Add some features, Test and improve Japanse translations, and send these patches to AICP.

Recently we have some issues about branding. Since Android 9.0 "Pie" based release, we have decided to rebrand it. Floko, it's our new home.

Features

FlokoROM based on AICP, but we have some difference like:

  • Kawaii Easter Egg (Akane Time!!)
  • Add OmniSwitch
  • Support Advanced Lockscreen pattern (4x4, 5x5, 6x6)
  • Use LineageOS's Gallery2 and chromium-webview package
  • Remove Phonograph (Music Player)
  • Remove "Send ROMstats"
  • Remove "Post log to Hastebin"

Changelog

How to Install

  1. Flash recommended firmware or stock rom
    • Find on xda-developers, other forums, ...
  2. Flash recommended TWRP image
    • Find on twrp.me, xda, ...
  3. TAKE BACKUP ALL YOUR DATA
  4. Wipe /system, /data, /cache, Dalvik/ART Cache
    • WARN: You'll lost all your userdata in device at this time
  5. Flash ROM, GApps, Magisk, ...
  6. Reboot to System
  7. Enjoy.
