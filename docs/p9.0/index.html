History

In 2017, we started a project called "AICP-mordiford". It's a fork of AICP (Android Ice Cold Project), and it is an UNOFFICIAL build.
We worked on n7.1(nougat) and o8.1(oreo) releases. We added some features, tested and improved Japanse translations, and sent these patches to AICP.

Recently we have had some issues about branding. Since our Android 9.0 "Pie" based release, we have decided to rebrand it. Floko, it's our new home.

Features

FlokoROM is based on AICP, but we have some differences like:

  • Kawaii Easter Egg (Akane Time!!)
  • Add OmniSwitch
  • Support Advanced Lockscreen pattern (4x4, 5x5, 6x6)
  • Use LineageOS's Gallery2 and chromium-webview package
  • Remove Phonograph (Music Player)
  • Remove "Send ROMstats"
  • Remove "Post log to Hastebin"

Changelog

You can see the changelog on your device. Open Settings → Extras → CHANGELOG.

How to Install

  1. Flash recommended firmware or stock rom
    • Find on xda-developers, other forums, ...
  2. Flash recommended TWRP image
    • Find on twrp.me, xda, ...
  3. MAKE A BACKUP OF ALL YOUR DATA
  4. Wipe /system, /data, /cache, Dalvik/ART Cache
    • WARN: YOU WILL LOSE ALL USEDATA DURING A FLASH AT THIS TIME
  5. Flash ROM, GApps, Magisk, ...
  6. Reboot to System
  7. Enjoy.
