History

In 2017, we started a project called "AICP-mordiford". It's a fork of AICP (Android Ice Cold Project), and it is an UNOFFICIAL build.
We worked on n7.1(nougat) and o8.1(oreo) releases. We added some features, tested and improved Japanese translations, and sent these patches to AICP.

Since our Android 9.0 "Pie" based release, we have decided to start our new project. Floko, it's our new home.

Features

FlokoROM v2.0 is based on crDroid, but we have some differences like:

  • Add OmniSwitch
  • Kawaii Easter Egg in Floko Settings
  • Remove Music Player
  • Remove Font Packages

...and there are crDroid and LineageOS related features:

  • SmartNav(SmartBar, Fling, Pulse...)
  • Network traffic monitor
  • switch 4G/LTE icon
  • Battery bar
  • Statusbar Quick/Smart pulldown
  • QS columns
  • Lockscreen customization
  • Switch "Recents" style
  • HW button configurations
  • Expanded desktop
  • Rounded corners
  • Style(Theming)
  • Volume panel customization
  • brand-new dashboard

Changelog

You can see the changelog on your device. Open Floko Settings → About FlokoROM → Current changelog.

How to Install

  1. Flash recommended firmware or stock rom
    • Find on xda-developers, other forums, ...
  2. Flash recommended TWRP image
    • Find on twrp.me, xda, ...
  3. MAKE A BACKUP OF ALL YOUR DATA
  4. Wipe /system, /data, /cache, Dalvik/ART Cache
    • WARNING: YOU WILL LOSE ALL USERDATA DURING A FLASH AT THIS TIME
  5. Flash ROM, GApps, Magisk, ...
  6. Reboot to System
  7. Enjoy.
